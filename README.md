# @femboy/printf

Admittedly useless package that aliases `console.log` to `printf`.

Used for testing publishing to JSR :owiehappy:

# Usage

```ts
import { printf } from "@femboy/printf";

printf("testing", "123");
```

It's just like `console.log`! ~~I wonder why...~
